2D T-Rex 网格划分

本次使用Pointwise软件划分一个2D NACA机翼模型的T-Rex网格，详细步骤如下所示

求解器设置

  1. CAE，Select Solver
  2. CGNS
  3. OK
  4. CAE,Set Dimension
  5. 2D

导入几何

  1. File，Import，Database
  2. 选择RAF89.iges
  3. Open
  4. OK

网格节点设置

  1. 点击Layers
  2. 点击Empty Layers
  3. 双击10号边界层的Description field
  4. 输入Airfoil Grid
  5. 点击Set Current，将10号边界层设为当前边界层
  6. 取消Show Empty Layers 选项卡

默认设置

  1. 点击Defaults选项卡
  2. 确保Connector选中
  3. 点击Average $\varDelta s$，输入0.05
  4. Max. Deviation中输入0.0005
  5. Begin $\varDelta s$中输入0.005
  6. End $\varDelta s$中输入0.005

创建连接点

  1. 点击List选项卡
  2. 选中Database中所有的曲线
  3. 点击工具栏中的Connectors on Database Entities
  4. 点击Defaults选项卡
  5. 点击Reset Defaults 按钮
  6. 选中Dimension 选项
  7. 输入31
  8. 点击工具栏 2 Point Curves 选项
  9. 点击Entity Type,Connector
  10. 选中点A
  11. 选中Advanced 选项卡
  12. XYZ Offset 中输入 5 0 0
  13. 选中点B
  14. XYZ Offset 中输入 0 5 0
  15. 选中点C
  16. XYZ Offset 中输入 -10 0 0
  17. 选中点D
  18. XYZ Offset 中输入 0 -10 0
  19. OK
  20. 选择连接线：BC和CD
  21. Ctrl+C
  22. Ctrl+V
  23. Mirror
  24. 选择并输入Y=0.0
  25. OK
  26. OK
  27. 选择spacing constraint
  28. 选择A点附近，输入0.005
  29. 选择B点附近，输入0.5
  30. 点击List选项卡中的全部connectors
  31. 点击选项卡中的Points On

创建非结构网格计算域

  1. 选择工具栏中的Unstructures
  2. 点击工具栏中Creat,Assemble Special,Domain
  3. 确保Auto Save 和Auto Complete 都不选中
  4. 选择AB连接线
  5. 选择机翼上层连接线
  6. 选择机翼下层连接线
  7. 再次选择AB连接线
  8. 选中Auto Save 和Auto Complete
  9. 选择BF
  10. OK

创建T-Rex 计算域

  1. 选择刚创建的计算域
  2. Grid, T-Rex
  3. Max. Layers 中输入25
  4. Full Layers 中输入5
  5. 在Cell Types 中选择Triangles and Quads
  6. 取消Use Default next to Growth Rate 选项
  7. Growth Rate中输入1.2
  8. 点击Boundary Conditions 选项卡
  9. 点击 New
  10. 双击new BC (bc-2)中的name，输入T-Rex
  11. 双击T-Rex 的Type，选择Wall
  12. 双击T-Rex 中的$\varDelta s$，输入0.005
  13. 选择AO和OA连接线
  14. 选择AB的两侧
  15. 选择T-Rex BC
  16. 点击New
  17. 双击new BC (bc-3)中的name，输入Symmetry
  18. 选择Symmetry中的Type field 为Match
  19. 选择BC 和 BF连接线
  20. 选择Symmetry BC
  21. 点击Solve 选项卡
  22. 点击Initialize
  23. OK

调节网格分布增长率

  1. 选择BC 和 BF连接线

  2. Grid, Distribute

  3. 点击 Spacing 选项卡

  4. 选择B 点附近的连接线

  5. Spacing中输入0.005

  6. 选择C和F的spacing constraints

  7. Spacing 中输入0.3

  8. 点击Functions 选项卡

  9. 选择Growth

  10. Begin 按钮中选择Number of Layers and Growth Rate

  11. Number of Layers 中输入25

  12. Growth Rate 中输入1.2

  13. Apply

  14. 点击Dimension 选项卡

  15. 选择BC 连接线

  16. 点击Set Optimal Dimension button

  17. 点击Select Connector button

  18. 选择BF 连接线

  19. 点击Set Optimal Dimension button

  20. Apply

  21. OK

保存项目文件

  1. 点击工具栏中的Save
  2. 输入文件名
