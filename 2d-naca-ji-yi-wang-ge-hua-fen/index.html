2D NACA机翼网格划分

本次使用Pointwise软件划分一个2D NACA机翼模型的网格，详细步骤如下所示

软件介绍

Pointwise是一个非常好用的网格划分软件，其有如下优点：

  1. 易于学习、使用灵活、接口丰富
  2. 手动和自动来提高网格质量，可控性强
  3. 自动装配面或体网格，也可拉伸创建网格
  4. 直接读取CAD文件，高效清理不完善模型
  5. 适用于非常复杂几何模型的T-Rex技术
  6. 可实现网格划分参数化功能的Glyph脚本

导入几何文件

  1. File, Import, Database
  2. 选择NACA6412.igs文件
  3. Open
  4. OK

修改默认设置

  1. 点击Defaults选项卡
  2. 确保Connector 前面的框被选中
  3. 在Dimension中输入75，然后回车

创建连接点（Connector）

  1. 点击List选项卡
  2. 选择database中所有的曲线
  3. 点击工具栏的Connectors on Database Entities选项
  4. 选择Connector中所有曲线
  5. 点击工具栏中的Points on选项，可得到如下图形

调节机翼尾端连接点之间的距离

  1. 选择机翼尾端的两条连接线

  2. 点击工具栏中Grid选项卡中Dimension选项

  3. 选中Average $\varDelta s$选项

  4. 输入0.005，依次点击Dimension、OK

调节控制点之间的距离

  1. 点击All Masks On/Off 按钮，仅保留Spacing Constraints mask 选项被勾选
  2. 选中机翼尾端上表面和下表面曲线，并输入0.0005
  3. 选中机翼首端上表面和下表面曲线，并输入0.005

创建计算域

  1. 勾选Connectors mask
  2. 选择所有的connectors
  3. 点击Create选项卡中的Extrude, Norma选项
  4. 点击Attributes 选项卡
  5. 勾选Step Size 选项
  6. 在Initial $\varDelta s$中输入0.0001
  7. 勾选Orientation 选项
  8. 若箭头方向没有朝外就点击Flip 按钮
  9. 点击Run 选项卡
  10. 输入91步
  11. 点击Run
  12. 点击 OK

保存项目

  1. 点击工具栏中的Save 按钮
  2. 在弹出的对话框中输入要保存的项目名即可
